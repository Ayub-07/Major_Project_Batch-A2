#  Indian Coin Denomination Recognition using Deep Learning

## 📌 Project Overview

This project implements a **Deep Learning–based system for recognizing Indian coin denominations** from images.
The system uses **Computer Vision and Convolutional Neural Networks (CNN)** to classify coins into their respective denominations.

The goal of this project is to demonstrate how **Artificial Intelligence can be used in automated currency sorting systems**, reducing manual effort and improving efficiency.

---

##  Features

* Detects and classifies **Indian coin denominations**
* Uses **Deep Learning (CNN)** for image classification
* Simple to run using a **single batch file**
* Includes **test images for demonstration**

##  Requirements

Make sure you have the following installed:

* **Python** (version specified in requirements)
* Required libraries listed in `requirements.txt`

Install dependencies using:

```
pip install -r requirements.txt
```

##  How to Run the Project

1️⃣ Clone the repository

```
git clone https://github.com/Ayub-07/Major_Project_A2.git
```

2️⃣ Open the project folder

3️⃣ Install required libraries

```
pip install -r requirements.txt
```

4️⃣ Run the project

Simply **double click**:

```

```

The system will start and allow you to test coin recognition.
